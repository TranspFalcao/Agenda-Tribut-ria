# Agenda Tributária - Transportadora Falcão Ltda

Este pacote contém os arquivos para publicação da agenda tributária no GitHub Pages:

- `index.html`: Página principal com tabela de vencimentos.
- `style.css`: Estilos responsivos e modernos.
- `logo_falcao_otimizada_transparente.png`: Logo otimizada com fundo transparente.
- `README.md`: Este arquivo de instruções.

## Publicação

1. Crie um repositório no GitHub.
2. Envie todos os arquivos.
3. Ative o GitHub Pages nas configurações.
4. Acesse o site pelo link gerado.
